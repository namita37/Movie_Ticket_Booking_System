The Movie Ticket Booking System is a Java-based desktop application developed using Java Swing for the user interface and MySQL database for backend data storage. The main aim of this system is to automate the movie ticket booking process in theaters and multiplexes and remove the problems of manual booking systems.
In traditional booking systems, users often face issues like long queues, slow seat selection, and chances of double booking. Managing ticket records manually is also time-consuming and error-prone for theater staff. This project solves these problems by providing a fast, digital, and user-friendly booking system.
The system allows users to select movies, choose show timings, and book seats using an interactive seat grid layout. Each seat is represented as a button, and users can easily select or deselect seats. The system ensures that already booked seats cannot be selected again, preventing duplication.
The project uses MySQL database to store all important data such as movie details, seat status, and booking history. This ensures permanent storage and accurate tracking of bookings. Users can also view their previous bookings through the history feature.
After successful booking, the system automatically generates a ticket file containing movie name, show time, selected seats, and total amount. This acts as an e-ticket for the user.
The system also includes a cancel booking feature, where users can cancel booked tickets and update seat availability in real time.
Key Features:
•	Movie and show time selection 
•	Seat selection using grid layout 
•	Real-time seat availability update 
•	Prevention of duplicate booking 
•	Ticket cost calculation 
•	Booking confirmation using dialog box 
•	Booking history using MySQL 
•	Automatic ticket file generation 
•	Cancel booking functionality 
This system provides a simple, efficient, and reliable way to book movie tickets and demonstrates the use of Java Swing, OOP concepts, and JDBC database integration.
